Book Operations

* = 'https://noahgallo-netcentric-final.herokuapp.com'

  • GET
    • */api/books/

      returns all the books as a json array.

      Including a Path Variable of "title" will return the book with the specified title

    • */api/books/:id

      returns the book of the specified id

  • POST
    • */api/books/

      Post with path variables in the following Format:

      • title = [string]
      • description = [string]
      • price = [number]
      • year = [number]
      • hardcover = [bool]
      • authors[0]['name'] = [string]
      • authors[0]['nationality'] = [string]
      • authors[1]['name']=[string]
      • authors[1]['nationality'] = [string]

      - Authors must start from 0 and must not skip Integers. You May add as many as needed, just keep increasing the number.

  • PATCH
    • */api/books/:id

      Use the same variable formats POST uses

      • title = [string]
      • description = [string]
      • price = [number]
      • year = [number]
      • hardcover = [bool]
      • authors[0] = [ID]
      • authors[1] = [ID]

      - Patching for Authors requires that you use the ID of an existing Author

  • Delete
    • */api/books/:id

      Deletes the specified ID
